Capabilities

13 Specialized AI Agents

Each agent is an expert in its domain, working in concert to deliver comprehensive data science analysis.

Planner

Detects task type — classification, regression, clustering, or time series — and creates an execution plan.

Profiler

Deep-dives into column types, missing values, duplicates, memory usage, and data quality metrics.

Cleaner

Smart imputation (mean, median, KNN), outlier capping, date conversion, and duplicate removal.

EDA

Comprehensive exploratory analysis with correlations, distributions, and target variable deep-dives.

Visualizer

Dark-themed charts including histograms, boxplots, heatmaps, and custom analysis plots.

Feature Engineer

Automatic encoding, scaling, and mutual information-based feature selection for optimal performance.

Modeler

Trains LogisticRegression, RandomForest, and XGBoost with rigorous cross-validation scoring.

HPO

Bayesian hyperparameter optimization with Optuna — 20 trials to find the best configuration.

Evaluator

Holdout metrics, confusion matrices, residual plots, and comprehensive model performance scoring.

Explainer

SHAP-based feature importance with summary and bar plots for transparent model interpretation.

Insight Generator

LLM-powered business insights using Gemini Flash, translating model results into actionable advice.

Reporter

Professional PDF report generation with all results, charts, metrics, and insights embedded.

Chat Q&A

RAG-powered conversational interface — ask questions about your analysis and get instant answers.
